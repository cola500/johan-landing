Hur jag jobbar

I praktiken handlar mitt arbetssätt ofta om att skapa klarhet, dialog och framdrift.

Förstå verkligheten

Jag börjar med att prata med rätt personer och försöka förstå hur situationen faktiskt ser ut – inte bara hur den beskrivs på papper.

Skapa dialog

Många problem handlar om att viktiga perspektiv inte kommer fram. Jag arbetar aktivt för att skapa forum där människor vågar bidra.
